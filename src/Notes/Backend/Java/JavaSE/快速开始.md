# 注释

注释是用于解释代码的文本，不会被编译器执行。Java 中有三种注释方式：

| 注释类型 | 语法              | 用途          |
| -------- | ----------------- | ------------- |
| 单行注释 | `// 注释内容`     | 单行注释      |
| 多行注释 | `/* 注释内容 */`  | 多行注释      |
| 文档注释 | `/** 注释内容 */` | 生成 API 文档 |

```java
// 单行注释

/*
 * 多行注释
 * 可以跨越多行
 */

/**
 * 文档注释
 * 用于生成 API 文档
 */
```

# 字面量

字面量是直接出现在代码中的常量值，代表固定的值。

| 类型   | 示例                             | 说明                             |
| ------ | -------------------------------- | -------------------------------- |
| 整数   | `100`, `0b1010`, `0o144`, `0x64` | 十进制、二进制、八进制、十六进制 |
| 浮点数 | `3.14`, `1.5e2`                  | 小数形式、科学计数法             |
| 字符   | `'A'`, `'\n'`, `'\u0041'`        | 单字符、转义字符、Unicode 字符   |
| 字符串 | `"Hello"`, `"\n"`                | 双引号包围的字符序列             |
| 布尔值 | `true`, `false`                  | 真、假                           |
| 空值   | `null`                           | 表示引用类型的空值               |

# 变量

变量是用于存储数据的容器，需要先声明后使用。

## 变量声明与初始化

```java
// 声明变量
int age;
String name;

// 声明并初始化变量
int age = 18;
String name = "Alice";

// 多个变量声明
int a, b, c;
int x = 1, y = 2, z = 3;
```

## 变量作用域

| 作用域   | 定义位置                 | 可见范围                       |
| -------- | ------------------------ | ------------------------------ |
| 局部变量 | 方法或代码块内部         | 仅在定义它的方法或代码块内可见 |
| 成员变量 | 类内部，方法外部         | 整个类内部可见                 |
| 静态变量 | 类内部，使用 static 修饰 | 整个类内部可见，可通过类名访问 |

# 标识符

标识符是用于命名变量、方法、类等的名称。

## 标识符命名规则

1. 由字母、数字、下划线 `_` 和美元符号 `$` 组成
2. 不能以数字开头
3. 不能使用 Java 关键字和保留字
4. 区分大小写

## 命名规范

| 类型   | 规范                         | 示例                         |
| ------ | ---------------------------- | ---------------------------- |
| 类名   | 驼峰命名法，首字母大写       | `Student`, `UserManager`     |
| 方法名 | 驼峰命名法，首字母小写       | `getAge()`, `calculateSum()` |
| 变量名 | 驼峰命名法，首字母小写       | `name`, `studentAge`         |
| 常量名 | 全部大写，单词间用下划线分隔 | `MAX_VALUE`, `PI`            |

# 数据类型

Java 是强类型语言，每个变量都必须指定数据类型。

## 基本数据类型

| 类型      | 大小   | 范围                 | 默认值   | 示例                |
| --------- | ------ | -------------------- | -------- | ------------------- |
| `byte`    | 1 字节 | -128 到 127          | 0        | `byte b = 100;`     |
| `short`   | 2 字节 | -32768 到 32767      | 0        | `short s = 1000;`   |
| `int`     | 4 字节 | -2^31 到 2^31-1      | 0        | `int i = 100000;`   |
| `long`    | 8 字节 | -2^63 到 2^63-1      | 0L       | `long l = 100000L;` |
| `float`   | 4 字节 | 3.4e-38 到 3.4e+38   | 0.0f     | `float f = 3.14f;`  |
| `double`  | 8 字节 | 1.7e-308 到 1.7e+308 | 0.0d     | `double d = 3.14;`  |
| `char`    | 2 字节 | 0 到 65535           | '\u0000' | `char c = 'A';`     |
| `boolean` | 1 位   | true 或 false        | false    | `boolean b = true;` |

## 引用数据类型

- 类：如 `String`, `ArrayList` 等
- 接口：如 `List`, `Map` 等
- 数组：如 `int[]`, `String[]` 等

## 类型转换

Java 中的类型转换分为自动类型转换和强制类型转换。

### 自动类型转换

自动类型转换是指从低精度类型到高精度类型的转换，不需要显式声明。

| 转换方向                                               | 示例                       |
| ------------------------------------------------------ | -------------------------- |
| `byte` → `short` → `int` → `long` → `float` → `double` | `int i = 100; long l = i;` |
| `char` → `int`                                         | `char c = 'A'; int i = c;` |

```java
// 自动类型转换示例
byte b = 100;
short s = b;      // byte → short
int i = s;        // short → int
long l = i;       // int → long
float f = l;      // long → float
double d = f;     // float → double

char c = 'A';
int i2 = c;       // char → int
```

### 强制类型转换

强制类型转换是指从高精度类型到低精度类型的转换，需要显式声明，可能会导致精度损失。

| 转换方向                                               | 示例                                |
| ------------------------------------------------------ | ----------------------------------- |
| `double` → `float` → `long` → `int` → `short` → `byte` | `double d = 3.14; int i = (int) d;` |
| `int` → `char`                                         | `int i = 65; char c = (char) i;`    |

```java
// 强制类型转换示例
double d = 3.14;
int i = (int) d;       // double → int，结果为 3，精度损失

long l = 1000000;
int i2 = (int) l;      // long → int

int i3 = 65;
char c = (char) i3;    // int → char，结果为 'A'

// 注意：强制类型转换可能会导致数据溢出
int i4 = 200;
byte b = (byte) i4;     // int → byte，结果为 -56，数据溢出
```

### 类型转换规则

1. **自动类型转换**：低精度类型可以自动转换为高精度类型
2. **强制类型转换**：高精度类型必须通过强制类型转换为低精度类型
3. **精度损失**：强制类型转换可能会导致精度损失或数据溢出
4. **布尔类型**：布尔类型不能与其他类型进行转换

# 关键字

关键字是 Java 语言中具有特殊含义的单词，不能用作标识符。

## 常用关键字

| 类别     | 关键字                                                                                         | 说明               |
| -------- | ---------------------------------------------------------------------------------------------- | ------------------ |
| 访问控制 | `public`, `protected`, `private`                                                               | 控制成员的访问权限 |
| 类与对象 | `class`, `interface`, `extends`, `implements`, `new`                                           | 定义和创建类与接口 |
| 数据类型 | `byte`, `short`, `int`, `long`, `float`, `double`, `char`, `boolean`, `void`                   | 基本数据类型       |
| 流程控制 | `if`, `else`, `switch`, `case`, `default`, `for`, `while`, `do`, `break`, `continue`, `return` | 控制程序流程       |
| 异常处理 | `try`, `catch`, `finally`, `throw`, `throws`                                                   | 处理异常           |
| 修饰符   | `static`, `final`, `abstract`, `synchronized`                                                  | 修饰类、方法、变量 |
| 其他     | `package`, `import`, `this`, `super`, `instanceof`, `enum`                                     | 包管理、对象引用等 |

# 运算符

运算符用于对操作数进行运算。

## 算术运算符

| 运算符 | 名称 | 示例             | 结果         |
| ------ | ---- | ---------------- | ------------ |
| `+`    | 加法 | `5 + 3`          | 8            |
| `-`    | 减法 | `5 - 3`          | 2            |
| `*`    | 乘法 | `5 * 3`          | 15           |
| `/`    | 除法 | `5 / 3`          | 1 (整数除法) |
| `%`    | 取模 | `5 % 3`          | 2            |
| `++`   | 自增 | `int a = 5; a++` | a 变为 6     |
| `--`   | 自减 | `int a = 5; a--` | a 变为 4     |

## 赋值运算符

| 运算符 | 示例     | 等价于      |
| ------ | -------- | ----------- |
| `=`    | `a = 5`  | `a = 5`     |
| `+=`   | `a += 5` | `a = a + 5` |
| `-=`   | `a -= 5` | `a = a - 5` |
| `*=`   | `a *= 5` | `a = a * 5` |
| `/=`   | `a /= 5` | `a = a / 5` |
| `%=`   | `a %= 5` | `a = a % 5` |

## 比较运算符

| 运算符 | 名称     | 示例     | 结果  |
| ------ | -------- | -------- | ----- |
| `==`   | 等于     | `5 == 3` | false |
| `!=`   | 不等于   | `5 != 3` | true  |
| `>`    | 大于     | `5 > 3`  | true  |
| `<`    | 小于     | `5 < 3`  | false |
| `>=`   | 大于等于 | `5 >= 3` | true  |
| `<=`   | 小于等于 | `5 <= 3` | false |

## 逻辑运算符

| 运算符 | 名称   | 示例              | 结果  |
| ------ | ------ | ----------------- | ----- |
| `&&`   | 逻辑与 | `true && false`   | false |
| `\|\|` | 逻辑或 | `true \|\| false` | true  |
| `!`    | 逻辑非 | `!true`           | false |

## 位运算符

| 运算符 | 名称       | 示例                  | 结果      |
| ------ | ---------- | --------------------- | --------- |
| `&`    | 按位与     | `5 & 3` (101 & 011)   | 1 (001)   |
| `\|`   | 按位或     | `5 \| 3` (101 \| 011) | 7 (111)   |
| `^`    | 按位异或   | `5 ^ 3` (101 ^ 011)   | 6 (110)   |
| `~`    | 按位取反   | `~5` (~101)           | -6        |
| `<<`   | 左移       | `5 << 1` (101 << 1)   | 10 (1010) |
| `>>`   | 右移       | `5 >> 1` (101 >> 1)   | 2 (10)    |
| `>>>`  | 无符号右移 | `5 >>> 1`             | 2         |

## 三元运算符

```java
// 语法：条件表达式 ? 表达式1 : 表达式2
int max = (a > b) ? a : b;
```

| 6      | `<`, `<=`, `>`, `>=`, `instanceof`                                         | 从左到右 |
| 7      | `==`, `!=`                                                                 | 从左到右 |
| 8      | `&`                                                                        | 从左到右 |
| 9      | `^`                                                                        | 从左到右 |
| 10     | `\|`                                                                       | 从左到右 |
| 11     | `&&`                                                                       | 从左到右 |

# 代码示例

## 变量与数据类型示例

```java
public class VariableDemo {
    // 成员变量
    int memberVar;
    // 静态变量
    static String staticVar = "Static Value";

    public static void main(String[] args) {
        // 局部变量
        int localVar = 10;
        System.out.println("局部变量: " + localVar);
        System.out.println("静态变量: " + staticVar);

        // 不同数据类型
        byte b = 100;
        short s = 1000;
        int i = 100000;
        long l = 1000000000L;
        float f = 3.14f;
        double d = 3.1415926;
        char c = 'A';
        boolean bool = true;

        System.out.println("byte: " + b);
        System.out.println("short: " + s);
        System.out.println("int: " + i);
        System.out.println("long: " + l);
        System.out.println("float: " + f);
        System.out.println("double: " + d);
        System.out.println("char: " + c);
        System.out.println("boolean: " + bool);
    }
}
```

## 运算符示例

```java
public class OperatorDemo {
    public static void main(String[] args) {
        // 算术运算符
        int a = 10, b = 3;
        System.out.println("a + b = " + (a + b));
        System.out.println("a - b = " + (a - b));
        System.out.println("a * b = " + (a * b));
        System.out.println("a / b = " + (a / b));
        System.out.println("a % b = " + (a % b));

        // 自增自减
        int c = 5;
        System.out.println("c++ = " + c++);
        System.out.println("++c = " + ++c);

        // 比较运算符
        System.out.println("a > b: " + (a > b));
        System.out.println("a == b: " + (a == b));

        // 逻辑运算符
        boolean x = true, y = false;
        System.out.println("x && y: " + (x && y));
        System.out.println("x || y: " + (x || y));
        System.out.println("!x: " + !x);

        // 三元运算符
        int max = (a > b) ? a : b;
        System.out.println("最大值: " + max);
    }
}
```
