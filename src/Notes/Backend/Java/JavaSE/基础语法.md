# 分支结构

## if-else 语句

if-else 语句是最基本的条件分支结构，根据条件判断执行不同的代码块。

```java
// 基本 if 语句
if (condition) {
    // 条件为真时执行
}

// if-else 语句
if (condition) {
    // 条件为真时执行
} else {
    // 条件为假时执行
}

// if-else if-else 语句
if (condition1) {
    // 条件1为真时执行
} else if (condition2) {
    // 条件2为真时执行
} else {
    // 所有条件都为假时执行
}
```

## switch 语句

switch 语句用于多条件分支，根据表达式的值选择执行对应的代码块。

```java
switch (expression) {
    case value1:
        // 表达式等于value1时执行
        break;
    case value2:
        // 表达式等于value2时执行
        break;
    default:
        // 表达式不等于任何case值时执行
        break;
}
```

# 循环结构

## for 循环

for 循环用于重复执行指定次数的代码块。

```java
// 基本 for 循环
for (int i = 0; i < 10; i++) {
    // 执行循环体
}

// 增强 for 循环（foreach）
for (ElementType element : array) {
    // 遍历数组或集合
}
```

## while 循环

while 循环在条件为真时重复执行代码块。

```java
// while 循环
while (condition) {
    // 条件为真时执行
}

// do-while 循环
// 至少执行一次循环体
do {
    // 执行循环体
} while (condition);
```

# 控制语句

## break 语句

break 语句用于终止循环或 switch 语句的执行。

```java
// 在循环中使用 break
for (int i = 0; i < 10; i++) {
    if (i == 5) {
        break; // 终止循环
    }
    System.out.println(i);
}

// 在 switch 中使用 break
switch (day) {
    case 1:
        System.out.println("星期一");
        break;
    case 2:
        System.out.println("星期二");
        break;
    default:
        System.out.println("其他");
        break;
}
```

## continue 语句

continue 语句用于跳过当前循环的剩余部分，直接开始下一次循环。

```java
// 使用 continue
for (int i = 0; i < 10; i++) {
    if (i % 2 == 0) {
        continue; // 跳过偶数
    }
    System.out.println(i); // 只打印奇数
}
```

## return 语句

return 语句用于从方法中返回值，并终止方法的执行。

```java
// 返回值
public int add(int a, int b) {
    return a + b; // 返回两数之和
}

// 无返回值（void）
public void printHello() {
    System.out.println("Hello");
    return; // 可选，用于提前结束方法
}
```

# 示例代码

## 分支结构示例

```java
// if-else 示例
int score = 85;
if (score >= 90) {
    System.out.println("优秀");
} else if (score >= 80) {
    System.out.println("良好");
} else if (score >= 60) {
    System.out.println("及格");
} else {
    System.out.println("不及格");
}

// switch 示例
int day = 3;
switch (day) {
    case 1:
        System.out.println("星期一");
        break;
    case 2:
        System.out.println("星期二");
        break;
    case 3:
        System.out.println("星期三");
        break;
    default:
        System.out.println("其他");
        break;
}
```

## 循环结构示例

```java
// for 循环示例
System.out.println("使用 for 循环打印 0-9:");
for (int i = 0; i < 10; i++) {
    System.out.print(i + " ");
}
System.out.println();

// while 循环示例
System.out.println("使用 while 循环打印 0-9:");
int j = 0;
while (j < 10) {
    System.out.print(j + " ");
    j++;
}
System.out.println();

// do-while 循环示例
System.out.println("使用 do-while 循环打印 0-9:");
int k = 0;
do {
    System.out.print(k + " ");
    k++;
} while (k < 10);
System.out.println();
```
