# 什么是类

类是 Java 中最基本的概念，它是一种模板或蓝图，用于描述具有相同属性和行为的对象集合。类定义了对象的结构和行为。

# 什么是对象

对象是类的实例，是根据类创建的具体实体。每个对象都有自己的状态（属性）和行为（方法）。

# 类的定义

```java
// 类的基本定义
public class Person {
    // 属性（成员变量）
    private String name;
    private int age;

    // 构造方法
    public Person() {
    }

    public Person(String name, int age) {
        this.name = name;
        this.age = age;
    }

    // 方法（成员方法）
    public void setName(String name) {
        this.name = name;
    }

    public String getName() {
        return name;
    }

    public void setAge(int age) {
        this.age = age;
    }

    public int getAge() {
        return age;
    }

    public void sayHello() {
        System.out.println("Hello, my name is " + name + ", I'm " + age + " years old.");
    }
}
```

# 类的组成部分

## 1. 成员变量（属性）

成员变量是类的属性，用于存储对象的状态。

- **访问修饰符**：控制变量的访问权限（private、default、protected、public）
- **数据类型**：变量的类型（int、String、boolean 等）
- **变量名**：变量的名称

## 2. 构造方法

构造方法用于创建对象时初始化对象。

- 构造方法名称与类名相同
- 构造方法没有返回值
- 可以有多个构造方法（重载）
- 如果没有定义构造方法，编译器会自动生成一个无参构造方法

## 3. 成员方法

成员方法是类的行为，用于定义对象可以执行的操作。

- **访问修饰符**：控制方法的访问权限
- **返回类型**：方法返回值的类型，无返回值时使用 void
- **方法名**：方法的名称
- **参数列表**：方法接收的参数
- **方法体**：方法的具体实现

# 对象的创建和使用

## 创建对象

```java
// 使用无参构造方法创建对象
Person person1 = new Person();

// 使用有参构造方法创建对象
Person person2 = new Person("Alice", 20);
```

## 使用对象

```java
// 调用setter方法设置属性
person1.setName("Bob");
person1.setAge(25);

// 调用getter方法获取属性
System.out.println("Name: " + person1.getName());
System.out.println("Age: " + person1.getAge());

// 调用成员方法
person1.sayHello();
person2.sayHello();
```

# 访问修饰符

Java 提供了四种访问修饰符，用于控制类、成员变量和成员方法的访问权限：

| 修饰符              | 同一个类 | 同一个包 | 子类 | 所有类 |
| ------------------- | -------- | -------- | ---- | ------ |
| private             | √        | ×        | ×    | ×      |
| default（无修饰符） | √        | √        | ×    | ×      |
| protected           | √        | √        | √    | ×      |
| public              | √        | √        | √    | √      |

## 访问修饰符的使用建议

- **成员变量**：通常使用 private，通过 getter 和 setter 方法访问
- **成员方法**：根据需要使用不同的访问修饰符
- **构造方法**：根据需要使用不同的访问修饰符

# this 关键字

this 关键字表示当前对象的引用，用于：

1. 区分成员变量和局部变量
2. 调用当前类的其他构造方法
3. 返回当前对象

```java
// 区分成员变量和局部变量
public void setName(String name) {
    this.name = name; // this.name表示成员变量，name表示局部变量
}

// 调用当前类的其他构造方法
public Person() {
    this("Unknown", 0); // 调用有参构造方法
}

// 返回当前对象
public Person getSelf() {
    return this;
}
```

# 静态成员

静态成员属于类而不是对象，包括静态变量和静态方法。

## 静态变量

```java
public class Student {
    // 静态变量
    public static String schoolName = "ABC School";

    // 成员变量
    private String name;
    private int age;

    // 构造方法和其他方法...
}

// 访问静态变量
System.out.println(Student.schoolName); // 直接通过类名访问
```

## 静态方法

```java
public class MathUtils {
    // 静态方法
    public static int add(int a, int b) {
        return a + b;
    }

    // 静态方法
    public static int multiply(int a, int b) {
        return a * b;
    }
}

// 调用静态方法
int sum = MathUtils.add(5, 3);
int product = MathUtils.multiply(5, 3);
```

## 静态代码块

静态代码块在类加载时执行，用于初始化静态变量。

```java
public class StaticExample {
    // 静态变量
    public static int count;

    // 静态代码块
    static {
        count = 0;
        System.out.println("Static block executed");
    }

    // 其他代码...
}
```

# 实例代码

## 完整的类定义和使用示例

```java
// 类的定义
public class Car {
    // 成员变量
    private String brand;
    private String model;
    private int year;

    // 静态变量
    public static int totalCars = 0;

    // 构造方法
    public Car(String brand, String model, int year) {
        this.brand = brand;
        this.model = model;
        this.year = year;
        totalCars++;
    }

    // Getter和Setter方法
    public String getBrand() {
        return brand;
    }

    public void setBrand(String brand) {
        this.brand = brand;
    }

    public String getModel() {
        return model;
    }

    public void setModel(String model) {
        this.model = model;
    }

    public int getYear() {
        return year;
    }

    public void setYear(int year) {
        this.year = year;
    }

    // 成员方法
    public void start() {
        System.out.println(brand + " " + model + " started.");
    }

    public void stop() {
        System.out.println(brand + " " + model + " stopped.");
    }

    // 静态方法
    public static int getTotalCars() {
        return totalCars;
    }
}

// 类的使用
public class CarTest {
    public static void main(String[] args) {
        // 创建对象
        Car car1 = new Car("Toyota", "Corolla", 2020);
        Car car2 = new Car("Honda", "Civic", 2021);

        // 使用对象
        car1.start();
        car2.start();

        System.out.println("Car 1: " + car1.getBrand() + " " + car1.getModel());
        System.out.println("Car 2: " + car2.getBrand() + " " + car2.getModel());

        // 访问静态变量
        System.out.println("Total cars: " + Car.totalCars);
        System.out.println("Total cars (via method): " + Car.getTotalCars());

        car1.stop();
        car2.stop();
    }
}
```

# 总结

类和对象是 Java 编程的核心概念：

- **类**：是一种模板或蓝图，定义了对象的属性和行为
- **对象**：是类的实例，具有具体的状态和行为
- **类的组成**：成员变量、构造方法、成员方法
- **访问修饰符**：控制访问权限（private、default、protected、public）
- **this 关键字**：表示当前对象的引用
- **静态成员**：属于类而不是对象，包括静态变量和静态方法

通过类和对象的概念，Java 实现了面向对象编程的基本思想，使代码更加模块化、可维护和可扩展。
