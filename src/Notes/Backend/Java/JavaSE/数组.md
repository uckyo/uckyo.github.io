# 什么是数组

数组是一种用于存储固定大小的同类型元素的容器。在 Java 中，数组是引用类型，它可以存储基本数据类型或对象。

# 数组的特性

- **固定长度**：数组一旦创建，长度不可改变
- **同类型元素**：数组中的所有元素必须是相同类型
- **连续内存**：数组元素在内存中是连续存储的
- **索引访问**：通过下标（索引）访问元素，索引从 0 开始
- **默认值**：未初始化的数组元素会被赋予默认值
  - 数值类型：0
  - 布尔类型：false
  - 引用类型：null

# 数组的初始化

## 1. 声明并初始化

```java
// 声明并初始化int数组
int[] arr1 = new int[5]; // 长度为5的int数组，元素默认为0

// 声明并直接初始化
int[] arr2 = {1, 2, 3, 4, 5}; // 长度为5的int数组，元素为1,2,3,4,5

// 声明并使用new关键字初始化
int[] arr3 = new int[]{1, 2, 3, 4, 5}; // 与上面等价
```

## 2. 多维数组初始化

```java
// 二维数组初始化
int[][] arr2D1 = new int[2][3]; // 2行3列的二维数组
int[][] arr2D2 = {{1, 2, 3}, {4, 5, 6}}; // 直接初始化二维数组
int[][] arr2D3 = new int[2][]; // 不规则二维数组，每行长度可以不同
arr2D3[0] = new int[3];
arr2D3[1] = new int[2];
```

# 数组的操作

```java
// 1. 创建（Create）数组
int[] arr = new int[5]; // 创建长度为5的int数组

// 2. 读取（Read）数组元素
int value = arr[0]; // 读取第一个元素
System.out.println(value);

// 3. 更新（Update）数组元素
arr[0] = 100; // 将第一个元素更新为100

// 4. 删除（Delete）数组元素
// Java中数组长度固定，无法直接删除元素，通常通过以下方式实现：

// 方式1：将元素设为默认值
arr[0] = 0; // 将第一个元素设为0（int的默认值）

// 方式2：创建新数组，复制除了要删除元素之外的所有元素
int[] newArr = new int[arr.length - 1];
int indexToDelete = 0;
for (int i = 0, j = 0; i < arr.length; i++) {
    if (i != indexToDelete) {
        newArr[j++] = arr[i];
    }
}
arr = newArr; // 引用新数组
```

# 数组的遍历

## 1. for 循环遍历

```java
// 使用for循环遍历数组
for (int i = 0; i < arr.length; i++) {
    System.out.println(arr[i]);
}
```

## 2. 增强 for 循环（foreach）遍历

```java
// 使用增强for循环遍历数组
for (int value : arr) {
    System.out.println(value);
}
```

## 3. 数组工具类遍历

```java
// 使用Arrays.toString()打印数组
import java.util.Arrays;
System.out.println(Arrays.toString(arr));

// 使用Arrays.deepToString()打印多维数组
System.out.println(Arrays.deepToString(arr2D));
```

## 4. 流遍历（Java 8+）

```java
// 使用流遍历数组
Arrays.stream(arr).forEach(System.out::println);

// 使用流进行操作
int sum = Arrays.stream(arr).sum();
int max = Arrays.stream(arr).max().getAsInt();
int min = Arrays.stream(arr).min().getAsInt();
```

# 数组的常见操作

## 数组复制

```java
// 数组复制
int[] source = {1, 2, 3, 4, 5};
int[] target = new int[source.length];

// 方式1：使用System.arraycopy()
System.arraycopy(source, 0, target, 0, source.length);

// 方式2：使用Arrays.copyOf()
int[] target2 = Arrays.copyOf(source, source.length);

// 方式3：使用Arrays.copyOfRange()
int[] target3 = Arrays.copyOfRange(source, 1, 4); // 复制索引1到3的元素
```

## 数组排序

```java
// 数组排序
int[] arr = {5, 3, 1, 4, 2};
Arrays.sort(arr); // 升序排序
System.out.println(Arrays.toString(arr)); // [1, 2, 3, 4, 5]
```

## 数组查找

```java
// 数组查找
int[] arr = {1, 2, 3, 4, 5};
int index = Arrays.binarySearch(arr, 3); // 二分查找，返回索引
System.out.println(index); // 2
```

# 总结

数组是 Java 中最基本的数据结构之一，具有以下特点：

1. **固定长度**：一旦创建，长度不可改变
2. **同类型元素**：只能存储相同类型的数据
3. **索引访问**：通过下标快速访问元素
4. **连续内存**：元素在内存中连续存储，访问效率高
5. **默认值**：未初始化的元素会被赋予默认值

数组的优缺点：

**优点**：

- 访问速度快，时间复杂度为 O(1)
- 结构简单，易于理解和使用
- 适合存储固定大小的数据集合

**缺点**：

- 长度固定，无法动态调整
- 插入和删除操作效率低，需要移动元素
- 只能存储同类型元素

在实际开发中，数组常用于：

- 存储固定数量的同类型数据
- 作为其他数据结构的基础
- 进行数值计算和统计
- 实现简单的算法

当需要动态调整大小或频繁进行插入删除操作时，建议使用 Java 集合框架中的 ArrayList 等类。
